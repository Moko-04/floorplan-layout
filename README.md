# 図面レイアウト — 簡易店内造作CAD

PDF図面（または画像）を背景に読み込み、**縮尺を合わせて実寸で造作・什器を配置**するブラウザツール。
イラレのCAD的な作業を、もっと手軽に行うことを目的にしています。

## 起動方法
- **必ず `起動.bat` をダブルクリック** → ブラウザで自動的に開きます（ローカルサーバー: http://localhost:8791）
- ※ `index.html` の直接ダブルクリック（file://）ではPDFが開けないことがあります。**PDFを使うときは必ず `起動.bat` から**開いてください。
- pdf.js は `vendor/` 内に同梱済みのため、**インターネット接続は不要**です。

## 使い方（3ステップ）
1. **「📄 PDFを開く」** … 図面PDFまたは画像を読み込む
2. **「📏 縮尺合わせ」** … 図面上で長さが分かっている2点をクリック → 実寸(mm)を入力
   （例：壁2000mm、ドア900mm、通り芯など）
3. **作図** … 什器を選んで配置、矩形(部屋)・線(壁)を描く。寸法はmmで自動表示。

> PDFが無くても、縮尺合わせの段階で適当な2点に「1000mm」等を入れれば、白紙から実寸作図できます。

## 操作
| 操作 | 内容 |
|---|---|
| マウスホイール | 拡大・縮小（カーソル中心） |
| Space + ドラッグ / 中ボタンドラッグ | 画面移動 |
| 選択ツールでドラッグ | 移動 / 角ハンドルでサイズ変更 / 緑ハンドルで回転 |
| `Del` | 選択を削除 |
| `R` / `Shift+R` | 選択を15°回転 |
| 矢印キー | グリッド単位で微移動 |
| `Esc` | 選択解除 / ツール解除 |
| `Ctrl+S` | 保存 |

## 機能
- PDF複数ページ対応（◀ ▶ でページ切替）。**ページを変えると縮尺は要・再設定**。
- グリッドスナップ（50〜1000mm）
- 什器プリセット（恵比寿造作案_72坪 を参考に整備）。3グループ＋カスタム寸法：
  - **席・テーブル**：ポーカー台(2130×1060)/ダーツ台/丸テーブルφ650/角テーブル/カウンター/椅子/スツール/ソファ/ロッカー
  - **厨房・什器**：二層シンク/一層シンク/コールドテーブル(大)/ガスコンロ&作業台/作業台/冷蔵ショーケース/製氷機/冷蔵庫/レジ台
  - **設備・その他**：WC(個室)/ドア/引き戸/スクリーン/TVモニタ
- 右上パネルで、選択中の名称・幅・奥行(mm)・色を直接編集
- **保存/読込**（layout.json）… 形状と縮尺を保存。読込時は同じPDFを開いておくと位置が合います
- **PNG書き出し**（図面＋レイアウトを画像化）

## ファイル構成
- `index.html` … 画面
- `app.js` … ロジック
- `_serve.js` … ローカル配信サーバー（起動.bat が使用）
- `起動.bat` … 起動用

## 什器寸法のカスタマイズ
什器の種類・サイズは `app.js` 冒頭の `FIXTURES` 配列（mm単位）を編集すれば追加・変更できます。
